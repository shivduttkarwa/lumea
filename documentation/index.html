Luméa — WordPress Theme

A premium beauty & skincare WooCommerce theme for modern cosmetic, wellness, and personal care brands.

Version 1.1.5 Author Shivdutt Karwa WordPress 6.8+ WooCommerce 10.0+

Overview

Luméa is a full-featured beauty e-commerce theme built around a canvas-based animated hero, editorial product slider, scroll-reveal animations powered by GSAP, and a complete WooCommerce storefront. It ships with:

  • Slide-out cart drawer with AJAX quantity updates
  • Wishlist stored in browser localStorage via Lumea Core plugin
  • 9 custom page templates (About, Contact, FAQ, Wishlist, Bundles, Ingredients, Shipping, Coming Soon, Blank)
  • 56+ Customizer settings across 9 sections
  • One-Click Demo Import with full content, menus, and WooCommerce page assignments
  • Translation-ready with a POT file included
  • Schema.org JSON-LD product markup on single product pages

Installation

Method A — WordPress Admin (Recommended)

  1. Log in to your WordPress admin dashboard.

  2. Go to Appearance → Themes → Add New → Upload Theme.

  3. Click Choose File, select the lumea.zip file, and click Install Now.

  4. After installation completes, click Activate.

  5. A plugin notification banner will appear at the top of the admin. Install and activate Lumea Core and WooCommerce when prompted.

Method B — FTP

  1. Unzip lumea.zip on your computer.

  2. Upload the extracted lumea/ folder to /wp-content/themes/ via FTP.

  3. In WordPress admin, go to Appearance → Themes and activate Luméa.

  4. Install Lumea Core and WooCommerce via the plugin notification banner.

One-Click Demo Import

The demo import replicates the live preview in one click — creating all pages, blog posts, menus, and WooCommerce page assignments automatically.

  1. Install and activate the free One Click Demo Import plugin (search for it under Plugins → Add New).

  2. Go to Appearance → Import Demo Data.

  3. Click Import next to Luméa Main Demo. The process takes 1–2 minutes — do not navigate away.

  4. After import completes, go to Appearance → Customize and replace the placeholder hero and section images with your own licensed photography.

Demo images are placeholder AI-generated images included for layout reference only. Replace all hero images, editorial slides, and product images with your own licensed photography before going live.

The after-import hook automatically sets:

  • Front page and Posts page under Settings → Reading
  • Permalink structure to /%postname%/
  • Primary and Footer navigation menus
  • WooCommerce page IDs (Shop, Cart, Checkout, My Account)
  • Core Customizer defaults (footer headline, CTA, copyright)

WooCommerce Setup

  1. Install WooCommerce via Plugins → Add New (or the plugin notification banner after activating the theme).

  2. Run the WooCommerce setup wizard to configure currency, payments, and shipping.

  3. WooCommerce creates its own pages automatically (Shop, Cart, Checkout, My Account). You do not need to create these manually.

  4. Go to WooCommerce → Settings → General and confirm the Shop page is set correctly.

Product Placement Flags

Luméa Core adds a Lumea metabox on the product edit screen with two checkboxes:

  • Show in Bestsellers section — displays the product in the Homepage Bestsellers carousel
  • Show in Latest Products section — displays the product in the Homepage Latest Products grid

Set the Front Page

  1. Create a page titled Home (content can be left blank — the front page template handles everything).

  2. Go to Settings → Reading and set Your homepage displays to A static page.

  3. Set Homepage to your Home page and Posts page to your Blog page.

Navigation Menus

  1. Go to Appearance → Menus and click create a new menu.

  2. Name it Primary Menu and add your desired pages (Shop, Ingredients, Bundles, About, Journal are the demo defaults).

  3. Under Menu Settings at the bottom, check Primary Menu and save.

  4. Create a second menu named Footer Menu, add footer pages, and assign it to Footer Menu location.

If you used the One-Click Demo Import, both menus are created automatically.

Translation

Luméa is fully translation-ready. All user-facing strings use the text domain lumea. A POT file is included at languages/lumea.pot.

  1. Open languages/lumea.pot in Poedit or the Loco Translate WordPress plugin.

  2. Create a translation file (e.g. lumea-fr_FR.po / lumea-fr_FR.mo).

  3. Place the .mo file in wp-content/languages/themes/ or in the theme's languages/ folder.

Support

Support is provided via the comments section on the ThemeForest item page. Before posting, please check this documentation and the FAQ section of the item page.

When reporting an issue, please include:

  • WordPress version and PHP version
  • WooCommerce version
  • List of active plugins
  • A description of the issue and steps to reproduce it
  • A link to the affected page (if your site is live)
Response time is typically 1–2 business days. Support covers theme functionality as described in this documentation. It does not include general WordPress support, plugin customisation, or modifications beyond the original theme design.
